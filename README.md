# blink-2leds-wokwi

Código de piscar dois leds usando simulador Wokwi a partir do exemplo-base de piscar um led

Projeto inicial: https://wokwi.com/projects/344891652101374548

Projeto modificado: https://wokwi.com/projects/458963109359320065

Projeto (Yuri): https://wokwi.com/projects/459020945665280001
OBS: No Wokwi não há comentários descritivos.

O código possui uma variável de tempo para manter a proporcionalidade entre as fases do semáforo. Basta trocar o tempo da variável para ajustar a temporização geral ou os multiplicadores dentro do delay para alterar a proporção.
